Topic 1 — Spreadsheet Modeling

Mathematical logic, formulas, functions, and conditional operations

Foundation   BODMAS — Order of Operations

Every formula starts with =. The computer follows BODMAS strictly — brackets first, then orders (powers), then ÷ and ×, then + and −.

BBrackets
OOrders (²)
DDivision
MMultiplication
AAddition
SSubtraction
❌ Flawed logic
=10+5*2  → result: 20 (multiplies first!)
✅ Correct logic
=(10+5)*2  → result: 30 (brackets first)

Functions   Core Aggregation Functions

FunctionSyntax ExampleWhat It Does
=SUM() =SUM(B2:B10) Totals all numbers in a continuous range. Never use =A1+B1+C1 on large datasets.
=AVERAGE() =AVERAGE(C1:C20) Calculates the mathematical mean of all selected cells.
=MAX() =MAX(D1:D50) Returns the highest value in the dataset.
=MIN() =MIN(D1:D50) Returns the lowest value in the dataset.
=COUNTIF() =COUNTIF(A1:A20,"Passed") Counts cells that match an exact condition (text or number).
=SUMIF() =SUMIF(B1:B10,"Hardware",C1:C10) Checks column B for a condition, then adds corresponding values in column C.
=IF() =IF(A1>120,"Allowed","Too Short") Returns one value if condition is TRUE, another if FALSE.
=VLOOKUP() =VLOOKUP(A2,$D$1:$E$20,2,FALSE) Searches a table for a value and returns data from a chosen column.

Key Concept   Absolute Referencing with $

When you drag a formula down, cell references shift automatically. Use $ to lock a cell — like locking the Tax Rate so every row uses the same one.

Without $ — BREAKS when dragged
=B1*E1  → becomes =B2*E2, then =B3*E3
E1 shifts → wrong tax rate!
With $ — LOCKED when dragged
=B1*$E$1  → becomes =B2*$E$1, then =B3*$E$1
E1 stays fixed → correct!
Rule: Put $ before the column letter AND the row number to lock both axes: $E$1. This is used in VLOOKUP to lock the lookup table.
📊 Spreadsheet Checklist
BODMAS: I know the order of operations and can use brackets to control calculations.
=SUM(B2:B10): Totals all values in a range.
=AVERAGE(C1:C20): Finds the mathematical mean.
=MAX() / =MIN(): Finds the highest or lowest value.
=COUNTIF(range, "condition"): Counts cells matching a text or number condition.
=SUMIF(range, "condition", sum_range): Checks one column, adds values from another.
=IF(test, "true_value", "false_value"): Makes a boolean True/False decision automatically.
Absolute Referencing ($E$1): Locks a cell so formulas don't break when dragged down.
=VLOOKUP(lookup, $table, col, FALSE): Searches a lookup table and returns matching data.
Formatting: Currency to 2 decimal places; merged headers; adjust column widths so no data is truncated.

Topic 2 — Vector Graphics (Inkscape)

Mathematical shapes, paths, nodes, boolean operations & SVG code

🔢

Vector Graphics

Made of mathematical coordinates — lines and points. Scale to any size without ever losing quality. Used for logos, crests, icons, and print.

🟦

Bitmap / Raster

Made of tiny coloured squares called pixels. Blow it up too big and it looks blurry and blocky. Used for photographs.

📐

Align & Distribute

  • Align — lines up edges or centers on an axis
  • Distribute — equal space between each shape
  • Group — many shapes move as one
  • Ungroup — edit individual pieces again

Path Tools   Boolean Operations — Combining Shapes

Don't draw complex shapes by hand. Use basic shapes and combine them mathematically via Path menu.

Union (Add)
Melts shapes into one outline
Difference (Cut)
Top shape cuts a hole in bottom
Intersection
Keeps only overlapping part

Paths & Nodes   How Vectors Are Built

Vectors are mathematical coordinates called Nodes, joined by lines called Paths. You edit them with the Node Tool.

Bezier Pen Tool: Click point-by-point to trace real images with straight lines. Then use the Node Tool to pull handles and bend lines to match organic curves.
Text on a Path: Select text + a circular path → Text > Put on Path. Essential for school crests and badges.
Linked Offset: Creates a thick, uniform border around complex shapes or text — exactly like a vinyl sticker or sports decal.

SVG Code   XML Vector Code Basics

Vector graphics are actually written in code. The computer reads XML/SVG to draw shapes inside an <svg> tag.

Drawing a Circle
<circle
  cx="50"   ← center X coordinate
  cy="50"   ← center Y coordinate
  r="40"    ← radius (size)
  fill="red"
  stroke="black"
  stroke-width="3" />
Drawing a Rectangle
<rect
  x="50"      ← start X position
  y="50"      ← start Y position
  width="200"
  height="100"
  fill="blue" />
Circle uses cx/cy + r (center + radius).
Rect uses x/y + width/height (top-left corner + size).
✏️ Vector Graphics Checklist
Vector vs Bitmap: Vectors = math (infinite scale, never blurry). Bitmaps = pixels (get blurry when enlarged).
Align & Distribute: Align lines up edges/centers. Distribute puts equal space between shapes.
Group / Ungroup: Group joins shapes into one object. Ungroup lets you edit individual parts.
Path > Union: Melts overlapping shapes into one unified outline.
Path > Difference: Top shape acts as a cookie cutter — cuts a hole in the bottom shape.
Corner Node: Sharp, jagged point — line handles act independently.
Smooth Node: Flowing curve — handles locked in a straight line like a seesaw.
Bezier Pen Tool: Click point-by-point to trace images; bend lines with Node Tool.
Text on a Path: Text > Put on Path wraps text around a circle. Used for badges and logos.
Linked Offset: Creates a thick, uniform border — replicates vinyl sticker / sports decal effect.
SVG <circle>: cx, cy = center position. r = radius. fill = inside colour. stroke = border colour. stroke-width = border thickness.
SVG <rect>: x, y = top-left corner. width + height = size. fill = colour.

Topic 3 — Web Authoring (HTML, CSS & JavaScript)

Structure, style, and behaviour — the three layers of every website

🏗️

HTML — Structure

HyperText Markup Language. The skeleton of the page. Uses tags like <h1>, <p>, <img> to define content. It is a markup language, not a programming language.

🎨

CSS — Style

Cascading Style Sheets. The paint, wallpaper, and furniture. Controls colours, fonts, spacing, and layout. Separated from HTML to keep code clean.

⚡

JavaScript — Behaviour

A programming language that makes pages interactive. It runs inside <script> tags. For this exam, you only need to know variables, alert(), and prompt().

HTML   Page Anatomy & Common Tags

Every HTML page has the same structure. The <head> holds invisible metadata; the <body> holds everything you see.

<html>
  <head>
    <title>My Page</title>           ← shows in browser tab
    <meta charset="UTF-8">         ← character encoding
    <meta name="description" content="...">
    <meta name="author" content="...">
    <link rel="stylesheet" href="style.css">  ← external CSS
  </head>
  <body>
    <h1>Main Heading</h1>          ← h1 to h3 available
    <p>A paragraph of text.</p>
    <img src="photo.jpg" alt="Description">
    <a href="https://...">Click me</a>
    <ul><li>Item 1</li><li>Item 2</li></ul>
    <div id="myBox">A container</div>
  </body>
</html>

CSS   How a CSS Rule Works

A CSS rule = Selector + Declaration Block. The selector targets an element. The block contains property: value pairs.

/* Selector → targets all <p> tags */
p {
  color: #215482;     ← text colour
  font-size: 16px;    ← text size
  font-family: Arial, sans-serif;
  background-color: #fef08a;
  padding: 10px;      ← space INSIDE the border
  margin: 15px;       ← space OUTSIDE the border
  border: 1px solid black;
}

/* Class selector — apply to any element */
.highlight {
  background-color: yellow;
  font-weight: bold;
}
Cascading rule: Inline styles > Internal style sheet > External style sheet. When multiple rules target the same element, the most specific one wins.

JavaScript   The Three Things You Need to Know

JavaScript lives inside <script> tags at the bottom of the <body>. For this exam, master just these three concepts.

📦

1. Variables

A named box that stores data in the computer's memory.

var studentName = "Aisha";
var score = 95;
💬

2. alert()

Pops up a message box on screen. The user reads it and clicks OK.

alert("Welcome to my page!");

// Can also display a variable:
alert("Hello, " + studentName);
⌨️

3. prompt()

Pops up an input box asking the user to type something. Saves the answer into a variable.

var name = prompt("What is your name?");
alert("Hello, " + name + "!");
How it all fits together: Put your JavaScript inside <script></script> tags at the bottom of the <body>. Use prompt() to get input, store it in a var, then display it with alert().
📋 Cambridge Note: Paper 3 does NOT examine JavaScript (behaviour layer). For the exam, focus on HTML structure and external CSS stylesheets linked with <link rel="stylesheet" href="...">.
🌐 Web Authoring Checklist
Three Layers: HTML = structure, CSS = style/presentation, JavaScript = behaviour/interactivity.
HTML document structure: <html> > <head> + <body>. Head = metadata. Body = visible content.
Meta tags in <head>: charset, description, keywords, author, viewport.
Common tags: <h1>–<h3> headings, <p> paragraph, <ul>/<ol>/<li> lists, <img src="" alt="">, <a href="">, <div>, <hr>.
Attributes: Extra info inside the opening tag in name="value" format (e.g. src, href, alt, id, class).
CSS Rule: selector { property: value; } — semicolons separate declarations.
CSS selectors: Tag selector (p) targets all <p> tags. Class selector (.highlight) targets class="highlight". ID selector (#myBox) targets id="myBox".
Box Model: padding = space INSIDE border. margin = space OUTSIDE border.
External CSS: Linked in <head> with <link rel="stylesheet" href="style.css">. Used for exam tasks.
JavaScript variable: var name = "value"; — a named container that stores data in memory.
alert("message"): Pops up a message box. Can display text or a variable: alert("Hello, " + name);
prompt("question"): Pops up an input box. Whatever the user types is saved: var name = prompt("Your name?");
Script tag: All JavaScript goes inside <script></script> at the bottom of <body>.
