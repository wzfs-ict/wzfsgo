Website Authoring (HTML & CSS)

Strict separation of structure (HTML) and presentation (CSS) is required. You must edit and refine code manually, as WYSIWYG editors will not generate perfect, exam-compliant code.

1. Head & Meta Tags (Structure & Setup)

The <head> section contains metadata that is not displayed on the page itself but is critical for the browser and search engines.

<head> <!-- Sets the title in the browser tab --> <title>Tawara Conservation</title> <!-- Syllabus 2026: Mandatory Meta Tags --> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <meta name="description" content="A website about local wildlife conservation"> <!-- Linking the external CSS --> <link rel="stylesheet" href="style.css"> </head>

2. Text & Structural Tags (Content Layer)

HTML structures content hierarchically. Use headings logically and format paragraphs and lists precisely.

HTML Code

<h1>Main Heading</h1> <h2>Sub Heading</h2> <p>This is a standard paragraph.</p> <ul> <li>Unordered (Bullet) List Item</li> <li>Second Item</li> </ul> <ol> <li>Ordered (Numbered) List Item</li> </ol>

3. Multimedia & Hyperlinks

You must know how to embed video with fallback text and controls. Images must always have alternate text. Links must direct the user accurately.

<!-- Image with Alternate Text (MANDATORY) --> <img src="images/logo.png" alt="Company Logo depicting a tree" width="200" height="100"> <!-- Video with Controls and Fallback --> <video width="600" height="400" controls autoplay muted loop> <source src="wildlife.mp4" type="video/mp4"> Your browser does not support the video tag. </video> <!-- Hyperlinks --> <a href="https://www.cambridgeinternational.org" target="_blank">Opens in new window</a> <a href="mailto:info@tawara.org?subject=Website Enquiry">Send us an Email</a>

4. CSS Styling (Presentation Layer)

CSS controls how HTML elements look. You will be required to write external stylesheets and define specific classes.

CSS Code (style.css)

/* Name, Centre, Cand No */ body { background-color: #f0f8ff; font-family: Arial, sans-serif; } table { border-collapse: collapse; margin-left: auto; margin-right: auto; } /* Defining a Class */ .highlight { color: #ff0000; font-weight: bold; }
