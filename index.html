Grade 3: Digital Foundations

Classroom Basics

Instructions:

ScratchJr: Code Basics

Sequence:

Arranging code blocks in the correct order. The computer reads the sequence from left to right.

Event / Trigger:

The action that starts the code sequence. Examples include tapping the Green Flag or tapping a character.

ScratchJr: Efficiency

Loop:

A block used to repeat a sequence of actions automatically. This makes code shorter and more efficient.

Number Parameter:

The number written on a block. It specifies how many times to loop or how far to move.

Makers Empire: 3D Basics

3D Shape:

A solid object. It has Width, Height, and Depth.

Workplane:

The flat grid area on the screen where you place and build 3D models.

Makers Empire: Tools

  • Move: Sliding an object up, down, left, or right on the workplane.
  • Resize: Modifying the object to make it larger or smaller.
  • Rotate: Turning an object around to view or attach different sides.

Grade 4: Animation & Logic

Classroom Basics

Instructions:

  • Read the definitions carefully.
  • Revise the vocabulary words highlighted in green.

ScratchJr: Program Logic

Sequence:

A precise list of instructions. If an instruction is incorrect or out of order, it is called a Bug.

Loop:

A block that repeats instructions. You must define the exact number of repetitions required to reach the objective.

ScratchJr: Event Triggers

Start on Bump:

An event block that starts a sequence only when two characters collide on screen.

Send Message:

The envelope block. Character A sends a colored message signal to Character B to trigger an action.

Piskel: Stop Motion Basics

Frame:

A single, still picture. Animations are created by displaying multiple frames in sequence.

Duplicate:

A tool used to make an exact copy of a frame, allowing you to make small changes incrementally.

Piskel: Animation Tools

FPS (Frames Per Second):

Controls the speed of the animation. A high number plays fast; a low number plays slow.

Onion Skinning:

A tool that displays a faded outline of the previous frame to guide the drawing of the next frame.

Grade 5: Systems & AI

Classroom Basics

Instructions:

  • Read the definitions carefully.
  • Revise the vocabulary words highlighted in purple.

Programming Foundations

  • Sequence: The required order of instructions.
  • Loop: Repeating code to maintain efficiency.
  • Event / Trigger: The action that starts the code. Grade 5 projects utilize Message Envelopes to synchronize multiple processes.

Physical Computing

Physical Computing:

Writing software code to control a physical hardware device, such as the Botley robot.

Command & Transmit:

Inputting a sequence of commands on a remote, and transmitting that data to the robot to execute.

Artificial Intelligence

Data:

Information (images, numbers, text) provided to a computer system so it can learn.

Pattern Recognition:

The process where AI analyzes data to find similarities, allowing it to classify objects or make decisions.

3D & Vector Graphics

  • Axes: Lines defining 3D space. X (Left/Right), Y (Up/Down), Z (Depth).
  • Grouping: Fusing different shapes into a single solid 3D object.
  • Vector: An image created using mathematical formulas, preventing blurriness when resized.
  • Path & Node: The Path connects points. The Node is an anchor point used to bend the path.
