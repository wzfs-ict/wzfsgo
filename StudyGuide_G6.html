Section 1

The Data Organizer — Spreadsheets

Spreadsheets are powerful tools used for organizing information, doing calculations instantly, and discovering patterns using charts.

Key Vocabulary

Cell

A single rectangular box where you enter data.

Column

A vertical group of data running top to bottom, labelled with letters (A, B, C).

Row

A horizontal line of data running left to right, labelled with numbers (1, 2, 3).

Cell Reference

The unique name of a cell — column letter + row number (e.g. A1, C3).

Formulas & Functions

⭐ The Golden Rule
Every spreadsheet formula must always begin with an equals sign (=).
🔁 Dynamic Referencing
Never type static numbers directly into a formula if you can use a cell reference instead — use =A1+B1 instead of =5+10. That way, if the input data changes, your results update automatically. This is called "What If" modelling.

Basic Math: add (+), subtract (-), multiply (*), divide (/).

Replication (Fill Handle): drag the small cross at the corner of a cell to copy a formula down other rows — the software automatically adjusts the cell references for you.

Functions Reference

FunctionWhat it does
SUMAdds a list of numbers together.
MAX / MINFinds the largest / smallest number in a list.
AVERAGECalculates the mean value.
COUNTACounts all cells in a range that are not empty.

Charts & Formatting

Use formatting to make data readable — e.g. bold headings, or Currency (£) number formats. Charts (bar charts, line graphs) help you see patterns and trends in your data at a glance.

Section 2

The Logic Builder — Scratch Programming

Programming is about giving the computer instructions to solve a problem or run a game.

Variables

Variable

A named piece of data stored in memory that holds a single value (like "Score" or "Timer").

Name vs. Value

The Name ("Score") stays the same, even when the Value inside it changes.

🚩 Set vs. Change
Use Set [variable] to 0 at the very start of a game (under the Green Flag event) to reset the score. Use Change [variable] by 1 to update the score during the game.

Programming Concepts

ConceptWhat it means
SequenceInstructions performed in order, from top to bottom.
Iteration (Loops)Blocks like repeat or forever that run the same instructions multiple times.
SelectionRunning code only IF a condition is true (if... then blocks) — e.g. checking if score equals 10 to level up.

Section 3

The Digital Sculptor — 3D Modelling (Tinkercad)

3D modelling lets you build digital objects using Constructive Solid Geometry (CSG) — combining simple shapes to make complex ones.

The 3D Workspace

3D

Three Dimensions — a 3D shape has height, width, and depth.

Workplane

The flat, blue gridded surface where you place your designs.

Perspective / ViewCube

Change your viewing angle (top, front, side) to check shapes are truly touching, not floating.

Group

The tool used to permanently combine two or more shapes into a single solid object.

Tools & Handles

⬜
White Corner Handles

Resize a shape in two dimensions at the same time.

⬛
Black Side Handles

Resize a shape in one dimension.

🔺
Black Cone-Shaped Handle

Lift or lower an object off the workplane (Z-axis).

↻
Curved Arrows

Rotate your object.

Combining & Hollowing

Placeholders (Holes): a shape used to subtract volume or cut a hole into a solid object.

⚠️ Crucial Rule
When hollowing out a container (like a desk tidy or jewel box), you must lift the hole shape by about 2mm using the black cone handle before grouping — otherwise your box will not have a floor!

Section 4

Web & Graphic Design — Canva

A good website or poster must be easy to read, visually pleasing, and designed for a specific group of people.

Design for Your Audience

👧 Kids

Bright colors, fun fonts, exciting pictures.

👩‍🏫 Parents / Teachers

Clean designs, neat fonts, easy-to-find information.

🎮 Gamers

Dark backgrounds, bright colors, action pictures.

Typography & White Space

The 2-Font Rule: don't use too many fonts — stick to a maximum of 2 to 3 fonts to keep your design clean.

White Space: the empty space around text and pictures. It helps your design "breathe" and makes it easier to read.

Safe Sourcing & Copyright

Copyright: you cannot just take any picture from Google.

✅ Safe to use

Pictures and graphics inside the Canva "Elements" tab — Canva has already checked the licenses, so they're 100% safe for school projects.

❌ Risky to use

Random images copied straight from a Google search — these may be protected by copyright.

If you must give credit, use a simple formula in a small text box at the bottom of your page: Photo by [Name] on [Website]

Website Navigation

Sitemap

The structure of your website (e.g. Home, About, Contact pages).

Hyperlinks

Navigation paths linking text or buttons so users can click between pages.

Previewing: always use the Desktop/Mobile preview buttons in Canva to check your design on both a computer and a phone screen before publishing.
